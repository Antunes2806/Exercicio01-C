Programa desenvolvido em linguagem C para realizar a contagem de ocorrências de um caractere em um arquivo texto.

- Leitura de arquivo texto.
- Pesquisa de caractere.
- Contagem de ocorrências.
- Tratamento de erro para arquivo inexistente.

```bash
gcc main.c -o exercicio01
```
